## Description
This is a consoloe app that creates and manages local SQLite database. It allows you to add, sort and delete everyday tasks.
To choose menu item just print corresponding number.
Supported data format for adding tasks is YYYY-MM-DD.
It was build with Python 3.8.2 SDK.
